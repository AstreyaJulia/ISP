# ISP (Isida). Информационно-справочная система суда.

### Итоговая сборка API v.1

#### Клиентская часть:

1. React.js;
2. Tailwind CSS;
3. Storybook для библиотеки компонентов;

#### Серверная часть:

1. PHP;

### Требования:

Серверной части для запуска нужен работающий веб-сервер на Apache v.2.4 + PHP версии не ниже 8.1, например, Open Server
Panel.

### Настройка:

* Настройка серверной части осуществляется изменением файла conection.php;

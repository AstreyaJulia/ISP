Freq DAILY
1. больше ничего не выбрано
  повторение каждый день

  DAILY+ interval
    повторение через интервал дней ежедневно

2. DAILY + byweekday = WEEKLY + byweekday
  повторение по выбранным дням недели

  DAILY + byweekday + interval	= WEEKLY + interval + byweekday
    повторение через интервал недель по выбранным дням недели

3. DAILY + byweekday + bymonth = MONTHLY + byweekday + bymonth
  повторение выбранные месяцы, в дни недели
